# 📱 スマホで使う (PWA)

YouTube 期間別人気動画ソーターは、Chrome 拡張版に加えて **PWA (Progressive Web App)** としても配布しています。iOS Safari / Android Chrome のホーム画面に追加すると、ネイティブアプリのように単独ウィンドウで起動できます。

> **公開URL**: <https://ryo722.github.io/youtube-period-sorter/app/>

スマホブラウザで上記 URL を開き、手順に従ってホーム画面に追加してください。

---

## できること / できないこと

| 機能 | 拡張版 (Chrome) | PWA (iOS / Android) |
|---|:---:|:---:|
| チャンネルの自動検出 (現在のタブ) | ✅ | ⚠️ Web Share Target 経由のみ |
| 期間 × 人気順の絞り込み | ✅ | ✅ |
| 並び替え (再生回数 / 高評価 / コメント / 投稿日) | ✅ | ✅ |
| 24h ローカルキャッシュ | ✅ | ✅ |
| CSV / TSV エクスポート | ✅ | ✅ |
| オフライン起動 (UI のみ。API 通信は要オンライン) | ❌ | ✅ |
| 「YouTube アプリの共有」から直接呼び出し | ❌ | ✅ (Android) |

PWA でも **YouTube Data API v3 のキー (無料)** が必須です。下の「APIキー取得」を参照してください。

## インストール手順

### iOS (Safari)

1. **Safari** で <https://ryo722.github.io/youtube-period-sorter/app/> を開く
   - Chrome / Firefox では「ホーム画面に追加」が現れません。**必ず Safari** を使ってください
2. 画面下部 (または上部) の **共有ボタン** (⬆️ の四角) をタップ
3. メニューを下にスクロール → **「ホーム画面に追加」** をタップ
4. 名前を確認して **「追加」** をタップ
5. ホーム画面に YouTube 期間ソーターのアイコンができます

> iOS では Safari 経由でしかホーム画面に追加できない仕様です (PWA の OS 制約)。

### Android (Chrome)

1. **Chrome** で <https://ryo722.github.io/youtube-period-sorter/app/> を開く
2. 画面右上の **⋮ (メニュー)** をタップ
3. **「アプリをインストール」** または **「ホーム画面に追加」** をタップ
4. 確認ダイアログで **「インストール」** をタップ
5. ホーム画面に追加され、初回タップで単独ウィンドウで起動します

> Android Chrome では URL を開いた直後に「インストール」のバナーが自動で出ることもあります。出た場合はそれをタップしても OK です。

## API キーの取得 (初回のみ)

PWA は YouTube Data API v3 を直接呼び出します。**自分の API キーを発行して設定** してください (拡張版と同じ手順)。

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセスし、プロジェクトを作成 (既存プロジェクトでも可)
2. **「APIとサービス」→「ライブラリ」** で **YouTube Data API v3** を検索 → 有効化
3. **「APIとサービス」→「認証情報」→「認証情報を作成」→「APIキー」**
4. キー作成時の設定:

   | 項目 | 設定値 |
   |---|---|
   | 名前 | 任意 (例: `youtubeSort-pwa`) |
   | このキーを使用してアクセスできる API | **「キーを制限」** を選択 → **YouTube Data API v3** のみにチェック |
   | アプリケーションの制限 | スマホ Safari / Chrome から呼び出すため **「なし」のまま** で動作確認推奨 |

5. 発行されたキーをコピー
6. PWA を開く → 画面下の **「APIキー設定」** リンクをタップ
7. キーを貼り付けて **「保存」**
8. **「テスト」** ボタンで疎通確認

> 無料枠は 1日 10,000 ユニット。50件取得で約 101 ユニット消費するので、毎日数十回叩いても余裕で収まります。

## 使い方

### 通常起動

1. ホーム画面の YouTube 期間ソーターアイコンをタップ
2. **チャンネル入力欄に `@handle` または `UC...` を貼り付ける** (拡張版と違いタブの自動検出はないので手動入力)
3. 期間と件数を選んで **「人気順で表示」**
4. 結果が表示される。並び替え・エクスポートは結果ページのヘッダーから

### YouTube アプリから「共有」で起動 (Android のみ便利)

PWA は **Web Share Target** に対応しているので、YouTube アプリの共有メニューに入り込みます。

1. YouTube アプリでチャンネルページ (例: `youtube.com/@mkbhd`) を開く
2. **「共有」** をタップ
3. アプリ一覧から **「YouTube 期間別人気動画ソーター」** を選ぶ
4. PWA がチャンネル名を自動入力した状態で起動 → 期間を選んで「人気順で表示」

> iOS Safari の共有メニューには PWA は登録されません (iOS 側の仕様)。iOS の場合はチャンネル URL をコピーしてから PWA を開いて貼り付けてください。

## トラブルシューティング

### 「ホーム画面に追加」が出てこない (iOS)

- Chrome や Firefox で開いていませんか? **Safari でないと出ません**
- すでに同じドメインを追加済みの可能性があります。ホーム画面で既存アイコンを探してください

### インストールできない / バナーが出ない (Android)

- Chrome の「シークレットモード」では PWA インストールが無効です。通常モードで開き直してください
- すでにインストール済みの場合、メニューに「アプリを開く」と表示されます

### API キーを設定したのに「402 / 403」エラー

- Google Cloud Console でキーの「アプリケーションの制限」が「なし」になっているか確認してください (HTTP リファラ制限を入れると PWA から弾かれます)
- 「YouTube Data API v3」が有効化されているか、Cloud Console の「APIとサービス → 有効な API」で確認

### キャッシュが消えない / 古い結果が出る

- 結果ページ右上の **「強制再取得」** ボタンを押すとキャッシュ無視で取り直します
- それでも直らない場合は、ブラウザの「サイトデータ削除」で `ryo722.github.io` のデータを消してください

### オフラインで起動はするが、検索すると「失敗」になる

- これは仕様です。UI のみオフラインで開ける設計で、YouTube API 呼び出しはオンラインが必須です

## プライバシー / セキュリティ

- API キーはあなたの端末の `localStorage` にのみ保存されます。サーバには送信しません
- API 呼び出しは Google (`googleapis.com`) のみ。他のサーバには通信しません
- ソースコードは [GitHub](https://github.com/Ryo722/youtube-period-sorter) で公開 (MIT)
- 詳細は [プライバシーポリシー](./PRIVACY.md) を参照
