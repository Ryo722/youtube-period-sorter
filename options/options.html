設定

YouTube Data API v3 キー

Google Cloud Console で発行した API キーを入力してください。
キーは このブラウザのローカルストレージに平文で保存されます (他の拡張機能や Web サイトからは読み取れませんが、デバイスを物理的に他者に渡す場合は注意してください)。

キャッシュ

検索結果は 24 時間このブラウザにキャッシュされます (最大 30 件)。
結果が古いと感じる時は、結果ページの「強制再取得」ボタンか、ここの「キャッシュを全削除」を使ってください。
※「キャッシュを全削除」はキャッシュのみを削除します。API キーを消すには上の「API キーを削除」を使ってください。

API キー取得手順

  1. Google Cloud Console にアクセスし、プロジェクトを作成（または既存プロジェクトを選択）。
  2. 「APIとサービス」→「ライブラリ」で YouTube Data API v3 を検索 → 有効化。
  3. 「APIとサービス」→「認証情報」→「認証情報を作成」→「APIキー」を選択。
  4. キー作成画面で表示される項目の設定:
    • 名前: 任意 (例: youtubeSort)
    • このキーを使用してアクセスできる API: 必須 ─「キーを制限」を選び、一覧から YouTube Data API v3 のみにチェック。
    • サービス アカウントを介して API 呼び出しを認証する: オフのまま (これは Vertex AI / Gemini 用)
    • アプリケーションの制限: 「ウェブサイト」を選択し、 chrome-extension://<拡張ID>/* を追加してください (必須推奨)。 これにより、万が一キーが漏洩しても本拡張機能の外からは利用できなくなります。 拡張 ID は chrome://extensions/ で確認できます (Chrome Web Store 公開版は ID が固定されますが、未パッケージ版は端末ごとに変わります)。
      ※ 動作確認のために一時的に「なし」で試すことは可能ですが、保存・テスト後は必ず制限を設定してください。
  5. 発行されたキーをこの欄に貼り付けて「保存」→「テスト」で疎通確認。
  6. キーは 誰にも共有しないでください。 漏洩時は Google Cloud Console から削除して再発行できます。 このブラウザでは API キーが 平文でローカル保存されるため、 共用 PC で使う場合は使用後に「API キーを削除」ボタンで削除することを推奨します。
